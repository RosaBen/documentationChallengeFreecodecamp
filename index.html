JavaScript

JavaScript was initially created to “make web pages alive”.

The programs in this language are called scripts. They can be written right in a web page’s HTML and run automatically as the page loads.

Scripts are provided and executed as plain text. They don’t need special preparation or compilation to run.

In this aspect, JavaScript is very different from another language called Java.

alert( 'Hello, world!' );

Summary

  • JavaScript was initially created as a browser-only language, but it is now used in many other environments as well.
  • Today, JavaScript has a unique position as the most widely-adopted browser language, fully integrated with HTML/CSS.
  • There are many languages that get “transpiled” to JavaScript and provide certain features. It is recommended to take a look at them, at least briefly, after mastering JavaScript.

Java

Java is a high-level, general-purpose, memory-safe, object-oriented programming language.

Java was designed by James Gosling at Sun Microsystems.

System.out.println("Hello, World!");

Principles

There were five primary goals in creating the Java language:

  1. It must be simple, object-oriented, and familiar.
  2. It must be robust and secure.
  3. It must be architecture-neutral and portable.
  4. It must execute with high performance.
  5. It must be interpreted, threaded, and dynamic.

Python

Python is a high-level, versatile programming language known for its readability and simplicity.

It is widely used in web development, data science, artificial intelligence, automation, and more. Python’s vast ecosystem of libraries makes it a popular choice for beginners and experts alike.

print("Hello, World!")

Key Points About Python:

  • Easy to learn & use – Simple syntax, beginner-friendly
  • Interpreted & dynamically typed – No need for compilation, flexible coding
  • Multi-purpose – Used in web development, AI, data science, automation, etc.
  • Extensive libraries – NumPy, TensorFlow, Pandas, Django, etc.
  • Cross-platform – Runs on Windows, macOS, Linux, etc.
  • Large community – Strong support and resources available

Rust

Rust is a high-performance and memory-safe programming language designed to prevent common memory errors found in C/C++.

It is widely used in system development, embedded programming, and applications requiring high performance. Its memory management model, without a garbage collector, makes it ideal for critical software.

println!("Hello, World!");

Key Points About Rust

  • Memory safety – Prevents memory leaks and errors without a garbage collector
  • High performance – As fast as C/C++
  • Ownership & Borrowing – Strict memory management to avoid errors
  • Safe concurrency – Facilitates parallel programming without data races
  • Rich ecosystem – Cargo (package manager), Crates.io (libraries)
  • Interoperability – Can integrate with C and WebAssembly
  • Reliability – Used in critical systems (OS, blockchain, browsers)

Angular

Angular is a TypeScript-based open-source framework for building dynamic web applications.

Developed and maintained by Google, it provides a powerful ecosystem for developing scalable and maintainable single-page applications (SPAs).

sayHello() {console.log("Hello from Angular!")}

Key Points About Angular

  • Component-based architecture – Modular and reusable UI components
  • Uses TypeScript – Enhances JavaScript with static typing and OOP features
  • Two-way data binding – Synchronizes data between model and view automatically
  • Dependency injection – Improves code maintainability and scalability
  • Powerful CLI – Angular CLI simplifies project setup and development
  • Built-in directives & services – Provides functionalities like loops, conditions, and HTTP handling
  • Cross-platform – Used for web, mobile (Ionic), and desktop applications
  • Strong ecosystem – Supported by Google, large community, and extensive documentation
